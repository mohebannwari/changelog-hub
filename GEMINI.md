# Gemini Code Assistant Context

## Project Overview

This is a Next.js project called "ChangeHub". Based on the file names and content, it appears to be a changelog management platform. The main page is a login page with options for email/password, Google, GitHub, and SSO authentication. The project is set up with TypeScript, Tailwind CSS, and ESLint.

## Building and Running

To get started with the project, you need to install the dependencies and then run the development server.

### Key Commands:

*   **Install Dependencies:**
    ```bash
    npm install
    ```

*   **Run Development Server:**
    ```bash
    npm run dev
    ```

*   **Build for Production:**
    ```bash
    npm run build
    ```

*   **Start Production Server:**
    ```bash
    npm run start
    ```

*   **Lint Files:**
    ```bash
    npm run lint
    ```

## Development Conventions

*   **Framework:** The project uses the [Next.js](https://nextjs.org/) framework.
*   **Language:** The project is written in [TypeScript](https://www.typescriptlang.org/).
*   **Styling:** The project uses [Tailwind CSS](https://tailwindcss.com/) for styling.
*   **Linting:** The project uses [ESLint](https://eslint.org/) for linting.
*   **Package Manager:** The project uses [npm](https://www.npmjs.com/) as the package manager.
